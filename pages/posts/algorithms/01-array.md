## 数组

数组是内存中是一段连续的存储空间；支持随机访问；关键：索引与寻址；

|        操作         | 时间复杂度 |
| :-----------------: | :--------: |
|       Lookup        |    O(1)    |
|       Insert        |    O(n)    |
|       Delete        |    O(n)    |
|  Append(push back)  |    O(1)    |
| Prepend(push front) |    O(n)    | -->
